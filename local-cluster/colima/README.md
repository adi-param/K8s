# Colima

Runs container runtimes on macOS and Linux in a lightweight VM, with an optional built-in Kubernetes (k3s). A small CLI-first alternative to Docker Desktop.

**When to use:** Lightweight local dev on macOS, CLI-driven, free Docker Desktop alternative with Kubernetes on demand.

**Prerequisites:** Colima (and the Docker CLI / kubectl).
