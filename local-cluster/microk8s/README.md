# MicroK8s

Canonical's lightweight Kubernetes, installed as a single snap package. Low overhead, with addons enabled through a simple command (dns, storage, ingress, dashboard).

**When to use:** Linux hosts, single-command install, edge and appliance-style deployments.

**Prerequisites:** snapd (Linux); also available on macOS/Windows via a bundled VM.
