# minikube

Creates a local cluster inside a VM or container (selectable driver). Beginner-friendly with a large addon ecosystem (ingress, dashboard, metrics-server) and good docs.

**When to use:** Getting started, exploring addons, single or small multi-node clusters.

**Prerequisites:** A driver such as Docker, HyperKit, or VirtualBox.
