# k3d

Runs k3s (a lightweight, certified Kubernetes distribution) inside Docker. Very fast to start, low on resources, and easy to run several clusters side by side.

**When to use:** Fast throwaway clusters, running multiple clusters at once, low-resource machines.

**Prerequisites:** Docker.
