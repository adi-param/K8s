# Rancher Desktop

An open-source desktop app that provides a k3s-based Kubernetes cluster plus a container runtime (containerd or dockerd), with a GUI for picking the Kubernetes version.

**When to use:** GUI-driven local development on Mac/Windows/Linux, a free Docker Desktop alternative.

**Prerequisites:** Rancher Desktop installed.

## Create

Launch Rancher Desktop, enable Kubernetes, and pick a version in Preferences.

## Delete

Disable Kubernetes in Preferences, or use **Troubleshooting → Factory Reset**.
