# k3s

A lightweight, certified Kubernetes distribution that runs directly on the host as a single binary. Strips out some optional components and defaults to a small footprint.

**When to use:** Edge and IoT, low-resource hosts, running Kubernetes directly on a Linux machine without Docker-in-the-middle.

**Prerequisites:** A Linux host (or VM).
