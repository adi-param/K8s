# Docker Desktop

Docker Desktop ships a built-in single-node Kubernetes that you enable with a checkbox. The simplest way to get a cluster on Mac or Windows if you already run Docker Desktop.

**When to use:** Simplest possible start on Mac/Windows, single-node only.

**Prerequisites:** Docker Desktop.

## Create

Settings → Kubernetes → **Enable Kubernetes** → Apply & restart.

## Delete

Settings → Kubernetes → disable, or use **Reset Kubernetes Cluster**.
