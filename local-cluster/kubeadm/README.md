# kubeadm

The official tool for bootstrapping a Kubernetes cluster from scratch on real or virtual machines. More manual than the others (you install a container runtime, CNI, etc.), but it mirrors how a production cluster is stood up.

**When to use:** Learning cluster internals, production-like multi-node setups on VMs.

**Prerequisites:** One or more Linux hosts/VMs, a container runtime, and a CNI plugin.
